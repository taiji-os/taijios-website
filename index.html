What is TaijiOS?

TaijiOS is a modern fork of InfernoÂ® Distributed Operating System (originally developed at Bell Labs). It brings powerful distributed computing capabilities to modern systems with updated build tools and 9front driver integration.

Modern UI via KRYON

Note: We use KRYON as the primary standard for building interfaces. Kryon simplifies the complexity of Limbo/Tk into a declarative, modern syntax that talks directly to 9P.

Building a distributed interface in TaijiOS is now as simple as defining a block:

// Kryon Easy UI Example
Window {
    title = "Hello"
    width = 200

    Center {
        Button {
            text = "Click Me"
            onClick = say_hello
        }
    }
}

Learn more at KryonLabs.com

Technical Specification

  • Limbo VM: Portable Dis bytecode runs on native hardware or hosted (Linux/BSD/macOS).
  • 9P: Every resource (CPU, file, device) is shared via the Styx protocol.
  • Reproducible: First-class NixOS integration for guaranteed build states.

Quick Start (NixOS)

# Clone the repository
git clone https://github.com/taiji-os/TaijiOS.git
cd TaijiOS

# Enter the build shell
nix-shell

# Build and materialize the WM
./run-wm.sh
